1. Untuk PDF reporting kita bisa menggunakan library composer ataupun css langsung

* Jika menggunakan css
1. Tambahkan @media print{ } pada css dan didalam kurung itulah kita men-style document pdf yang kita butukan
2. Untuk ceknya, bisa klik kanan > print






* Jika menggunakan composer
1. Tambahkan & install depedency composer bernama mpdf
2. Dan documentasi lengkap di dalamnya/websitenya
